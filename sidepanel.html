FormBatcher
Batch-fill any web form · CSV / XLSX
使用说明（MVP）
不需要系统开发权限，模拟你在网页上输入与点击。
  1. 打开目标系统的“新增/编辑”表单页面。
  2. 打开浏览器侧边栏：点击扩展图标 → 打开侧边栏。
  3. 选择 CSV / XLSX 文件（第一行是表头）。
  4. 在“选择列”里选中列名，点击“开始点选网页字段”，然后去网页上点对应输入框完成绑定。
  5. 点击“只填当前行”，逐行完成录入。
提示：保存/提交由用户手动完成，避免误操作。
