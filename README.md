# FormBatcher MVP (Chrome 扩展)

## 你会得到什么
- 一个可安装的 Chrome 扩展（Manifest V3）
- 支持 **CSV 批量填表**（半自动字段映射 + 可选自动提交/下一条）
- 站点级配置保存（同一网页路径下复用映射）

> 注意：为保证包体积小，本 MVP **未内置 XLSX 解析库**。Excel 可通过“另存为 CSV”后导入。

## 安装（开发者模式）
1. 打开 Chrome → 输入 `chrome://extensions`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目的 `src` 目录

## 使用方法
1. 打开目标系统的“新增/编辑”表单页面
2. 点击浏览器扩展图标（或扩展列表）→ 会自动打开 Side Panel
3. 在 Side Panel 中：
   - 选择 CSV 文件（第一行是表头）
   - 在「选择列」中选一列 → 点击「开始点选网页字段」→ 回到网页点击对应输入框
   - 重复绑定多个列
   - （可选）勾选「自动提交」，并点选一次“保存/提交按钮”
   - （可选）勾选「自动下一条」，并点选一次“新增/下一条按钮”
4. 点击「开始批量填充」
5. 可随时「暂停/继续」或「停止并重置」
6. 点击「导出结果 CSV」导出成功/失败清单

## 常见问题
- 如果页面有验证码/弹窗确认/校验提示：建议关闭自动提交，改为半自动（填完你手动保存）。
- 如果点选后定位不准：请重新点选绑定；或尽量点输入框本体（不要点外层装饰元素）。
